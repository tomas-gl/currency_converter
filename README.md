# Currency_Converter

# API Folder:
## Install depedencies:

_composer install_

## Migrate tables to database:

_php artisan migrate_

## Fill tables with fake datas:

_php artisan migrate:fresh --seed_

## Start project:

_php artisan serve_


# admin Folder:
